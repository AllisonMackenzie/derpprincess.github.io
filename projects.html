Pokemon Info App

https://pokemoninfoapp.netlify.app/

React | JavaScript | RESTful API

Pokemon Info App is, as you guessed it, an app to display info about Nintendo/Game Freaks Pokemon. The idea is simple in nature, it uses fetch calls to https://pokeapi.co/ to list Pokemon via mapping the data (with Pagination) from the axios calls. Right side app of App lists basic general information about the Pokemon.
