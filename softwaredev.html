Neko Bot

StarLight

Software not currently available for download. However, an older version of the code is on GitHub

Core Java

StarLight was something I made while still in university to understand how building a 2D Game Engine from the ground up would be. Needless to say, it was very difficult.

The rendering engine primarily uses Graphics2D. The core logic is handled on a tick based system, similar to those found in RuneScape or MineCraft.

The Engine comes with a prototype example already and although the prototype is basic, it shows proof of concepts of an engine that is complex it nature, not to the complexity of things like Unreal Engine of course but the amount of math that goes on was not something I was expecting going into making this.

The Engine works by reading map files based on color hex coordinates that one could make in any art program and that coordinate runs through the engines logic to place blocks, monsters, people, etc.

One aspect of the engine I wanted was the ability for easily adding variable health display options, as can be seen here this is the "Hearts" option, similar to what you would see in The Legend of Zelda game series.

TAS Display

Not currently available to the public.

Lua

TAS, short for Tool Assisted Speedrun, is a method of creating a nearly perfect video game speedrun that humans alone could never accomplish. Essentially, it's recording your gameplay inputs and redoing everything as close to perfect.
TAS Display is my codename for the various Lua contributions I'm making to help make creating these types of video game movies easier. Essentially, it's wrapped inside of an executable that provide a lua file to be ran based on the game you're playing, if one exists inside the wrapper. I'm not sure of the extent to which I will be adding games to the list, but it currently supports over 10 games!
Information is wrapped in a Lua script that is executed in BizHawk, that shows various information on screen that is super important like a character's health and XYZ Position in 3D space.

More projects will be added soon!
