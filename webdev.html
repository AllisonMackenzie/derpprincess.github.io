No Glitches Allowed

https://www.noglitchesallowed.org/

React | TypeScript | JavaScript | RESTful API | Express | MongoDB | Node

No Glitches Allowed is a video game speedrunning marathon charity event hosted on https://www.twitch.tv under the SpeedRunsLive channel (https://www.twitch.tv/SpeedRunsLive). I am both part of the group who runs the marathon and am commissioned for the Lead Web Developer role on all things NOGA. We have raised over $12,000 for Go Rescue, a no kill animal rescue service in Norfolk, Virginia!

Pokemon Info App

https://pokemoninfoapp.netlify.app/

React | JavaScript | RESTful API

Pokemon Info App is, as you guessed it, an app to display info about Nintendo/Game Freaks Pokemon. The idea is simple in nature, it uses fetch calls to https://pokeapi.co/ to list Pokemon via mapping the data (with Pagination) from the axios calls. Right side app of App lists basic general information about the Pokemon.
